## 六、维护指南

### 1. 如何添加新产品

方法一：通过后台管理页面
1. 打开 admin.html
2. 登录（admin / 123456）
3. 点击"产品管理"标签
4. 点击"新增产品"按钮
5. 填写产品信息并保存

方法二：直接修改 localStorage（开发者调试用）
```javascript
const products = JSON.parse(localStorage.getItem('products'));
products.push({
  id: Date.now(),  // 使用时间戳生成唯一 ID
  brand: 'bsyy',
  brandName: '毕生优养',
  name: '新产品',
  ingredients: '配料说明',
  price: 99,
  badge: '新品',
  icon: '🌿',
  bgClass: 'p-bg-1'
});
localStorage.setItem('products', JSON.stringify(products));
```

### 2. 如何修改轮播图
1. 打开 admin.html
2. 登录后进入"轮播管理"
3. 点击"编辑"按钮修改内容
4. 保存后前台自动更新

### 3. 如何修改网站样式

#### 修改颜色主题
在 index.html 的 CSS 中修改 :root 变量：
```css
:root {
  --warm-cream: #FAF6F1;      /* 暖米色 */
  --light-cream: #F5F0E8;     /* 浅米色 */
  --soft-beige: #E8E0D5;      /* 柔和米色 */
  --gold: #C9A96E;            /* 金色 */
  --light-gold: #D4B97A;      /* 浅金色 */
  --warm-brown: #8B7355;      /* 暖棕色 */
  --bean-green: #A8C69F;      /* 豆绿色 */
  --light-green: #C5DDB8;     /* 浅绿色 */
  --pale-green: #E8F0E0;      /* 淡绿色 */
  --text-dark: #3D3D3D;       /* 深色文字 */
  --text-light: #7A7A7A;      /* 浅色文字 */
  --white: #FFFFFF;           /* 白色 */
}
```

#### 修改产品背景样式
修改或添加以下类：
```css
.p-bg-1 { background: linear-gradient(180deg, #F5F0E8 0%, #E8E0D5 100%); }
.p-bg-2 { background: linear-gradient(180deg, #E8F0E0 0%, #C5DDB8 100%); }
.p-bg-3 { background: linear-gradient(180deg, #F0EDE8 0%, #E0D8D0 100%); }
```

### 4. 如何修改品牌图标
在 HTML 中找到对应位置，修改 emoji 图标：
- 毕生优养：🌾
- 豆养坊：🫘
- 宋康优泉：💧

### 5. 如何清除缓存
如遇到数据不更新问题：
1. 打开浏览器开发者工具（F12）
2. 在控制台输入：
   ```javascript
   localStorage.clear();
   location.reload();
   ```

## 九、注意事项

1. 数据存储在浏览器 localStorage 中，清除浏览器数据会导致数据丢失
2. 建议定期备份产品和轮播图数据
3. 生产环境建议修改后台默认密码
4. 如需持久化数据存储，建议接入后端数据库
