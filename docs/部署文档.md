## 1. 项目整体结构说明

```

├── frontend/          # 微信小程序前端（Taro框架）
```

## 2. 服务器环境要求

### 2.1 操作系统要求

- **推荐系统**：Linux CentOS 7+ 或 Ubuntu 20.04+
- **内存要求**：至少 2GB RAM（推荐 4GB+）
- **硬盘空间**：至少 20GB 可用空间

### 2.2 必需软件版本

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| Node.js | 18.x LTS | 后端服务运行环境 |
| npm | 8.x+ | Node.js 包管理器 |
| Nginx | 1.18+ | Web 服务器、反向代理 |
| PM2 | 最新版 | Node.js 进程管理器 |

### 2.3 安装命令

```bash
# 安装 Node.js 18.x（CentOS）
curl -fsSL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs

# 安装 Node.js 18.x（Ubuntu）
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 安装 PM2
npm install -g pm2

# 安装 Nginx（CentOS）
yum install -y nginx

# 安装 Nginx（Ubuntu）
apt install -y nginx
```

## 3. 前端部署说明

### 3.1 微信小程序部署

#### 步骤一：修改 API 配置

文件路径：`frontend/config/index.js`

```javascript
// 生产环境配置（修改为实际服务器地址）
const prod = {
  env: {
    NODE_ENV: '"production"'
  },
  defineConstants: {
    API_BASE: JSON.stringify('https://api.bisheng-youyang.com')  // 修改为实际API地址
  }
}
```

#### 步骤二：编译小程序

```bash
cd D:\bisheng-youyang\frontend

# 安装依赖
npm install

# 编译微信小程序
npm run build:weapp
```

#### 步骤三：导入微信开发者工具

1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择目录：`D:\bisheng-youyang\frontend\dist`
4. 填入 AppID（在微信公众平台获取）
5. 点击「确认」

#### 步骤四：配置 appid

文件路径：`frontend/config/index.js`

找到 `WECHAT_APPID` 配置位置，填入你的小程序 AppID。

### 3.2 H5 官网部署

#### 步骤一：编译 H5 版本

```bash
cd D:\bisheng-youyang\frontend

# 编译 H5 版本
npm run build:h5
```

#### 步骤二：Nginx 配置示例

```nginx
# /etc/nginx/conf.d/bisheng-youyang.conf

server {
    listen 80;
    server_name www.bisheng-youyang.com;  # 修改为实际域名

    # H5 静态文件
    location / {
        root /var/www/bisheng-youyang/frontend/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # API 反向代理
    location /api/ {
        proxy_pass http://127.0.0.1:3000/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # 启用 HTTPS（推荐）
    # listen 443 ssl;
    # ssl_certificate /path/to/cert.pem;
    # ssl_certificate_key /path/to/key.pem;
}
```

#### 步骤三：启动 Nginx

```bash
# 测试配置
nginx -t

# 重启 Nginx
systemctl restart nginx

# 设置开机自启
systemctl enable nginx
```

### 3.3 域名配置说明

| 环境 | 域名 | 用途 |
|------|------|------|
| 开发环境 | api-dev.bisheng-youyang.com | 开发环境 API |
| 生产环境 | api.bisheng-youyang.com | 生产环境 API |
| H5 官网 | www.bisheng-youyang.com | H5 商城官网 |

**配置步骤：**
1. 在域名服务商添加 DNS 解析
2. A 记录指向服务器 IP
3. 等待 DNS 生效（通常 10 分钟 - 24 小时）

## 4. 后端部署说明

### 4.1 环境变量配置

在 `backend/` 目录下创建 `.env` 文件：

```bash
# API 服务配置
API_PORT=3000
API_HOST=http://localhost
NODE_ENV=production

# 数据库配置
DB_PATH=./database/shop.db

# JWT 配置（生产环境请修改为强密码）
JWT_SECRET=your-strong-jwt-secret-key-here
JWT_EXPIRES_IN=7d

# 微信小程序配置（从微信公众平台获取）
WECHAT_APPID=wx_your_appid_here
WECHAT_SECRET=your_app_secret_here
```

### 4.2 PM2 启动配置

在 `backend/` 目录下创建 `ecosystem.config.js`：

```javascript
module.exports = {
  apps: [{
    name: 'bisheng-youyang-api',
    script: 'src/app.js',
    instances: 1,
    autorestart: true,
    watch: false,
    max_memory_restart: '500M',
    env: {
      NODE_ENV: 'production',
      API_PORT: 3000
    }
  }]
}
```

### 4.3 启动后端服务

```bash
cd D:\bisheng-youyang\backend

# 安装依赖
npm install

# 启动服务（PM2）
pm2 start ecosystem.config.js

# 查看运行状态
pm2 status

# 查看日志
pm2 logs bisheng-youyang-api

# 设置开机自启
pm2 startup
pm2 save
```

### 4.4 数据库文件位置

| 文件 | 路径 | 说明 |
|------|------|------|
| SQLite 数据库 | `backend/database/shop.db` | 主数据库文件 |
| 初始化数据 | `backend/database/init.js` | 数据库初始化脚本 |
| 配置目录 | `backend/config/` | 后端配置文件 |

### 4.5 接口地址配置说明

| 环境 | 接口基础地址 |
|------|--------------|
| 开发环境 | `http://localhost:3000` |
| 生产环境 | `https://api.bisheng-youyang.com` |

前端需在 `frontend/config/index.js` 中配置对应的 API 地址。

## 5. 后台管理部署说明

### 5.1 构建后台管理系统

```bash
cd D:\bisheng-youyang\admin

# 安装依赖
npm install

# 构建生产版本
npm run build
```

构建完成后，静态文件输出到 `dist/` 目录。

### 5.2 Nginx 部署配置

```nginx
# /etc/nginx/conf.d/bisheng-admin.conf

server {
    listen 80;
    server_name admin.bisheng-youyang.com;  # 后台管理域名

    root /var/www/bisheng-youyang/admin/dist;
    index index.html;

    # SPA 路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 代理（如果后台需要调用后端API）
    location /api/ {
        proxy_pass http://127.0.0.1:3000/;
        proxy_set_header Host $host;
    }

    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

### 5.3 部署步骤

```bash
# 1. 上传构建文件到服务器
scp -r D:\bisheng-youyang\admin\dist/* user@server:/var/www/bisheng-youyang/admin/

# 2. 重启 Nginx
nginx -t && systemctl restart nginx
```

## 6. 小程序后台配置清单

### 6.1 登录微信公众平台

访问：https://mp.weixin.qq.com

使用小程序账号登录后台。

### 6.2 服务器域名白名单配置

进入「开发」→「开发管理」→「开发设置」

| 域名类型 | 填写内容 | 示例 |
|----------|----------|------|
| request 合法域名 | API 服务器域名 | `https://api.bisheng-youyang.com` |
| uploadFile 合法域名 | 文件上传域名 | `https://api.bisheng-youyang.com` |
| downloadFile 合法域名 | 文件下载域名 | `https://api.bisheng-youyang.com` |

**注意：**
- 域名必须备案
- 必须使用 HTTPS
- 每月有修改次数限制（普通小程序 5次/月）

### 6.3 类目配置

进入「设置」→「基本设置」→「基本信息」

选择小程序类目：
- **主类目**：医疗健康
- **辅类目**：保健养生/医疗器械（根据实际业务选择）

### 6.4 App Secret 安全设置

进入「开发」→「开发管理」→「开发设置」

1. **重置 AppSecret**（首次使用需重置）
2. **妥善保管 AppSecret**，不要泄露
3. 生产环境 AppSecret 只在服务端使用，不要暴露在前端代码中

### 6.5 版本号规范

| 字段 | 规范 | 示例 |
|------|------|------|
| 版本号格式 | 主版本.次版本.修订号 | 1.0.0 |
| 提审递增规则 | 每次提审 +0.0.1 | 1.0.0 → 1.0.1 |
| 大版本更新 | 重大功能迭代 +1.0.0 | 1.0.0 → 2.0.0 |

### 6.6 HTTPS 证书要求

- 证书类型：DV / OV / EV 均可
- 证书域名：需包含主域名（如 `*.bisheng-youyang.com`）
- 推荐使用 Let's Encrypt 免费证书
- 证书有效期：90天（Let's Encrypt），需配置自动续期

## 7. 数据库说明

### 7.1 SQLite 数据库

**文件位置：** `D:\bisheng-youyang\backend\database\shop.db`

**备份方法：**

```bash
# 登录服务器后执行
cd /path/to/bisheng-youyang/backend/database

# 备份数据库（推荐每日执行）
cp shop.db shop.db.backup.$(date +%Y%m%d)

# 定期清理过期备份（保留最近30天）
find . -name "shop.db.backup.*" -mtime +30 -delete
```

**备份文件恢复：**

```bash
# 停止服务
pm2 stop bisheng-youyang-api

# 恢复备份
cp shop.db.backup.20240101 shop.db

# 重启服务
pm2 restart bisheng-youyang-api
```

### 7.2 MySQL 迁移步骤（如需）

如需从 SQLite 迁移到 MySQL：

1. **安装 MySQL**
   ```bash
   # CentOS
   yum install -y mysql-server
   # Ubuntu
   apt install -y mysql-server
   ```

2. **创建数据库和用户**
   ```sql
   CREATE DATABASE bisheng_youyang CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   CREATE USER 'bisheng'@'localhost' IDENTIFIED BY 'your_password';
   GRANT ALL PRIVILEGES ON bisheng_youyang.* TO 'bisheng'@'localhost';
   FLUSH PRIVILEGES;
   ```

3. **修改后端配置**
   - 更新 `.env` 文件
   - 修改数据库连接配置
   - 修改数据库模型文件（从 SQLite 适配到 MySQL）

4. **数据迁移**
   - 使用 Navicat 或其他工具导出 SQLite 数据
   - 导入到 MySQL

## 8. 常见问题FAQ

### Q1: 小程序提示"合法域名校验失败"

**原因：** API 域名未加入 request 合法域名白名单

**解决方案：**
1. 登录微信公众平台
2. 进入「开发」→「开发管理」→「开发设置」
3. 在「request 合法域名」中添加 API 服务器域名
4. 保存并重新编译小程序

---

### Q2: PM2 启动后服务无响应

**排查步骤：**

```bash
# 1. 查看 PM2 日志
pm2 logs bisheng-youyang-api

# 2. 检查端口占用
netstat -tlnp | grep 3000

# 3. 检查 Node.js 版本
node -v

# 4. 手动启动测试
cd /path/to/backend
node src/app.js
```

---

### Q3: Nginx 502 Bad Gateway

**原因：** 后端服务未启动或端口配置错误

**解决方案：**
1. 确保后端服务正在运行：`pm2 status`
2. 检查 Nginx 配置中的 proxy_pass 地址是否正确
3. 重启 Nginx：`systemctl restart nginx`

---

### Q4: 小程序无法正常登录

**排查步骤：**
1. 检查微信公众平台 AppID 和 AppSecret 配置
2. 检查后端 `/api/auth/login` 接口是否正常
3. 检查前端 `API_BASE` 配置是否正确
4. 查看浏览器/小程序控制台错误信息

---

### Q5: 数据库文件损坏

**预防措施：**
1. 定期备份数据库文件
2. 使用 `pm2 stop` 停止服务后再备份

**恢复方法：**
```bash
# 使用最新备份恢复
cp shop.db.backup.20240101 shop.db
pm2 restart bisheng-youyang-api
```

---

### Q6: HTTPS 证书过期

**解决方案：**
1. 使用 Let's Encrypt 自动续期：
   ```bash
   # 安装 certbot
   yum install -y certbot python3-certbot-nginx

   # 申请证书
   certbot --nginx -d api.bisheng-youyang.com

   # 设置自动续期
   certbot renew --dry-run
   ```

2. 或联系证书服务商续期

---

### Q7: 前端编译报错"找不到模块"

**解决方案：**
```bash
# 清除缓存并重新安装
cd frontend
rm -rf node_modules package-lock.json
npm install
npm run build:weapp
```

---

### Q8: 后台管理页面空白

**排查步骤：**
1. 检查浏览器控制台是否有 JS 报错
2. 检查 Nginx 是否正确配置了 SPA 路由
3. 检查静态文件是否正确上传到服务器
4. 检查 API 代理配置是否正确

---

### Q9: API 接口返回 500 错误

**排查步骤：**
```bash
# 1. 查看后端日志
pm2 logs bisheng-youyang-api

# 2. 检查数据库文件是否存在
ls -la backend/database/

# 3. 检查环境变量配置
cat backend/.env
```

---

### Q10: 微信开发者工具导入失败

**解决方案：**
1. 确保已安装微信开发者工具
2. 确认 dist 目录中有编译好的小程序代码
3. 检查 AppID 是否正确
4. 尝试点击「导入项目」时选择"不导入 cloud"
