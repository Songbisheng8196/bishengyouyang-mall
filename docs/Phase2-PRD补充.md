## 一、Phase2 核心目标

Phase1 完成了全站框架搭建，所有模块均为"有框架无数据"状态。Phase2 的任务是把真实内容填充进去，让产品可售、课程可看、测评可用。

## 二、三大内容模块详细说明

### 2.1 产品素材导入（Track A）

**数据来源：** 用户自有产品实拍图片 + 配套文案素材（可直接入库）

#### 2.1.1 商品品类规划（参考 Phase1 PRD）

| 品类编号 | 品类名称 | 首批 SKU 数量 | 备注 |
|---------|---------|-------------|------|
| C01 | 花养膏系列 | 3-5个规格 | 核心产品线 |
| C02 | 养生花茶系列 | 5-8个单品 | 搭配花养膏销售 |
| C03 | 节气养生礼盒 | 2-3个套装 | 礼品场景 |

#### 2.1.2 商品数据字段要求

每个 SKU 必须填写以下字段：

| 字段 | 说明 | 必填 |
|------|------|------|
| name | 商品名称 | ✅ |
| categoryId | 所属品类 | ✅ |
| subtitle | 商品副标题 | ✅ |
| description | 商品详情（富文本） | ✅ |
| price | 售价（元） | ✅ |
| originalPrice | 划线价（可不填） | ○ |
| stock | 库存数量 | ✅ |
| images | 商品图片列表（≤9张） | ✅ |
| tags | 标签（体质适配/功效关键词） | ✅ |
| weight | 商品重量（g，用于计算运费） | ✅ |
| status | 上架状态 | ✅ |

#### 2.1.3 配套文案素材使用说明

用户提供的文案素材需按以下规则处理：
- 产品功效描述 → 填入 `description` 字段（富文本）
- 产品卖点 → 填入 `tags` 字段
- 配套养生知识 → 转为"每日养生"文章（Article模块）
- 宣传语/主图文案 → 用于首页 Banner

#### 2.1.4 图片导入规范

- 图片格式：JPG/PNG，建议 800×800px 以上
- 主图命名：`{商品ID}_main_{序号}.jpg`
- 详情图命名：`{商品ID}_detail_{序号}.jpg`
- 图片存储路径：`backend/uploads/products/`
- 图片 CDN 路径配置到 `backend/config/index.js`

#### 2.1.5 商品详情页合规要求（强制执行）

每个商品详情页必须包含以下固定文案（由前端框架占位，现填充真实商品后需确认合规文案仍然显示）：

> ⚠️ 本品为普通食品，不具有疾病预防、治疗或保健功能，不代替药物。
> 体质特殊者请咨询专业人士后食用。

#### 2.1.6 配套内容生成

在商品导入同时，生成配套"每日养生"文章：
- 每款产品生成 1 篇配套文章（功效解读/食用方法/体质适配）
- 文章挂在商品详情页"相关阅读"区域
- 文章由 AI 辅助起草，用户审核后发布

### 2.2 体质测评题目生成（Track B）

#### 2.2.1 题目设计原则

- 基于《中医体质分类与判定》国家标准（GB/T 12346-2006）
- 结合女性养生特殊需求（妇科调理、生理期、产后恢复等维度）
- 每题 4 个选项，1-4 分制
- 总分 → 9 种体质分类（平和质/气虚质/阳虚质/阴虚质/痰湿质/湿热质/血瘀质/气郁质/特禀质）

#### 2.2.2 20 题题目分布

| 题号 | 维度 | 权重体质 |
|------|------|---------|
| 1-2 | 总体精力/疲乏感 | 气虚质 |
| 3-4 | 畏寒/手脚冰凉 | 阳虚质 |
| 5-6 | 口干/便秘/失眠 | 阴虚质 |
| 7-8 | 体型/易胖/痰多 | 痰湿质 |
| 9-10 | 面部长痘/口苦/烦躁 | 湿热质 |
| 11-12 | 肤色/瘀斑/疼痛 | 血瘀质 |
| 13-14 | 情绪/焦虑/抑郁 | 气郁质 |
| 15-16 | 过敏/鼻炎/湿疹 | 特禀质 |
| 17-18 | 睡眠/大便/脉象 | 平和质（作为基准） |
| 19-20 | 女性专项（生理期/气血） | 气血综合 |

#### 2.2.3 测评结果与推荐方案绑定

每种体质 → 推荐食补方案：

| 体质 | 推荐产品 | 推荐食疗方 | 推荐课程 |
|------|---------|---------|---------|
| 气虚质 | 花养膏·红枣枸杞味 | 红枣山药粥 | 太极养生基础 |
| 阳虚质 | 花养膏·姜枣桂圆味 | 生姜红茶 | 冬日温阳食谱 |
| 阴虚质 | 花养膏·桑葚黄精味 | 银耳百合羹 | 清热润燥指南 |
| 痰湿质 | 花养膏·薏米茯苓味 | 薏米红豆粥 | 祛湿健脾课 |
| 湿热质 | 花养膏·菊花决明味 | 绿豆汤 | 清热解毒课 |
| 血瘀质 | 花养膏·玫瑰红花味 | 玫瑰花茶 | 活血化瘀课 |
| 气郁质 | 花养膏·佛手香橼味 | 玫瑰陈皮茶 | 疏肝解郁课 |
| 特禀质 | 花养膏·灵芝孢子味 | 灵芝茶 | 固本培元课 |
| 平和质 | 花养膏·基础款 | 均衡膳食 | 日常养生课 |

#### 2.2.4 AI 起草流程

1. mini-product 根据体质分类框架起草 20 道题目初稿
2. AI 辅助优化表述（通俗易懂，避免医学专业术语）
3. 用户审核确认题目内容
4. 题目数据录入 `backend/database/questions.json`
5. 后端接口 `GET /api/constitution/questions` 返回题目列表
6. 前端问卷页替换占位数据

### 2.3 AI 视频课程生成（Track C）

#### 2.3.1 课程内容规划

首批生成 10 门课程，分为三类：

| 课程类型 | 数量 | 定价 | 说明 |
|---------|------|------|------|
| 免费科普课 | 3门 | 免费 | 体质自测/花养膏介绍/养生基础 |
| 会员专享课 | 5门 | 会员免费 | 花茶搭配/生理期养护/四季食补/体质调理/节气养生 |
| 配套课程 | 2门 | 付费配套 | 购买产品解锁，配合产品销售 |

#### 2.3.2 单门课程素材包

每门课程需生成以下内容：

| 内容项 | 说明 | 时长/数量 |
|--------|------|---------|
| 视频 | AI 生成讲解视频（Seedance 2.0） | 3-5分钟/门 |
| 课程封面 | AI 生成配套封面图 | 1张 |
| 课程介绍 | 文字版课程简介 | 200字 |
| 章节讲义 | 视频章节摘要 | 3-5章节 |
| 配套产品 | 课程关联商品 | 1-3个SKU |

#### 2.3.3 AI 视频生成流程

1. **生成课程脚本** → AI 辅助起草课程讲解脚本（500-800字）
2. **生成配套图** → AI 生成课程封面图 + 章节插图
3. **生成视频** → Seedance 2.0 根据脚本生成讲解视频
4. **视频上传** → 上传至 CDN/视频平台，获取播放链接
5. **课程数据录入** → 后台录入课程信息 + 视频 URL
6. **前端接入** → 前端播放器接入真实视频 URL

#### 2.3.4 视频合规要求

所有视频必须在明显位置展示：
- 视频开头 3 秒：固定片头（含"普通食品科普内容，非医疗诊疗"声明）
- 视频下方：滚动字幕或固定水印免责文案

## 四、Phase2 合规总则

### 4.1 产品宣传合规（所有商品详情页强制执行）

- 商品功效描述仅使用"食补"、"调理"、"日常养生"等词汇
- 禁止出现"治疗疾病"、"药用价值"、"医疗效果"等词汇
- 每个商品详情页底部固定显示合规免责声明
- 不得在产品页面提及任何疾病名称

### 4.2 视频内容合规

- 所有视频需在显著位置声明"普通食品科普内容，非医疗诊疗"
- 视频内容不得涉及疾病治疗、处方用药
- 视频字幕/简介中不得出现医疗承诺

### 4.3 测评合规

- 测评结果页必须包含"本测评仅作日常膳食参考，非医疗诊断，身体不适请及时就医"
- 不得在结果页推荐任何具有医疗功效的产品
